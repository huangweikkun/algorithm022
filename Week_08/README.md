#### 位操作

```
左移: << （移位补零）
右移: >>  (移位补零）
按位或：| 存在1则结果为1，否则为0
按位与：& 两者都为1则结果为1，否则为0
按位取反：~ 每个位取相反值
按位异或：^ 相同为0不同为1,也可理解为不进位加法

```

##### 按位异或常见操作

```
x^0 = x
x^1s = ~x 1s = ~0;
x^(~x) = 1s
x^x = 0
c = a ^ b => a^c=b,b^c=a
a^b^c = a^(b^c) = (a^b)^c
```

##### 常见位运算操作

1. 将 x 最右边的 n 位清零：x & (~0 << n)
2. 获取 x 的第 n 位值（0 或者 1）： (x >> n) & 1
3. 获取 x 的第 n 位的幂值：x & (1 << n)
4. 仅将第 n 位置为 1：x | (1 << n)
5. 仅将第 n 位置为 0：x & (~ (1 << n))
6. 将 x 最高位至第 n 位（含）清零：x & ((1 << n) - 1)
7. x & 1 == 1 (奇数) x & 1 == 0 (偶数)
8. x >> 1 => x/2
9. x = x&(x-1) 清零最低位的1
10. x & -x => 得到最地位的1
11. x & ~x => 0
